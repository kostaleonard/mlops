mlops.republication.republication.republish(versioned_object_path: str, republication_path: str, version: str) → str

Saves the versioned dataset files to the given path. If the path and appended version already exists, this operation will raise a PublicationPathAlreadyExistsError.

Parameters

  • versioned_object_path – The path to the published versioned object.

  • republication_path – The path, either on the local filesystem or in a cloud store such as S3, to which the object should be saved. The version will be appended to this path as a subdirectory. An S3 path should be a URL of the form “s3://bucket-name/path/to/dir”. It is recommended to use this same path to publish all datasets, since it will prevent the user from creating two different datasets with the same version.

  • version – The version of the versioned object.

Returns

The versioned object’s republication path.
